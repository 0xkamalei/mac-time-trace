# Product Overview

This is a native macOS time tracking application built with SwiftUI and SwiftData, similar to the Timing app. The application automatically tracks application usage and provides manual timer functionality for focused work sessions.

## Core Features

- **Automatic Activity Tracking**: Records app usage automatically through system notifications
- **Hierarchical Project Management**: Supports nested projects with drag-and-drop reordering
- **Manual Time Entries**: Users can create time entries and assign them to projects
- **Timeline Visualization**: Interactive timeline showing activities, projects, and time entries
- **Activity Analysis**: Detailed views of app usage with filtering and grouping capabilities

## Target Users

- Freelancers who need accurate time tracking for billing
- Professionals seeking to improve productivity and time management
- Teams requiring project time tracking while maintaining privacy
- Anyone wanting to understand their Mac usage patterns

## Key Value Propositions

- Privacy-focused: Data stored locally with optional sync
- Automatic tracking reduces manual effort and improves accuracy
- Hierarchical project structure supports complex workflows
- Rich visualization helps users understand time allocation
